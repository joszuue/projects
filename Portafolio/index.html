Proyectos

Mi Tienda System

Mi Tienda System" es una plataforma de comercio electrónico desarrollada en PHP con el patrón de arquitectura MVC. Permite gestionar productos y ventas en línea: añadir, editar, eliminar productos, controlar inventario, categorizar y procesar pagos. Ideal para centralizar operaciones y mejorar la experiencia del cliente.
ELEGANT

Elegant es una tienda en línea desarrollada en PHP con arquitectura MVC, que permite a los usuarios registrarse, explorar productos y agregar artículos a su carrito de compras de manera sencilla y eficiente para una experiencia de compra completa.
EcoSystem

Sistema de encuestas dinámicas creadas a partir de un estándar de calificación que toma en cuenta las diversas percepciones que el trabajador tiene de las estructuras y procesos en un medio laboral, refiriéndose al ambiente de trabajo de la organización. Fue desarrollado con Java y los frameworks JSF y PrimeFaces.
Veterinaria

El proyecto "Veterinaria" es una aplicación web diseñada para la gestión de una clínica veterinaria, implementada como un sistema CRUD (Create, Read, Update, Delete) desarrollado con PHP, Laravel y Filament. Este proyecto permite administrar de manera eficiente datos relacionados con pacientes, dueños, citas y tratamientos.
Sistema de Gestión de Personas

Proyecto CRUD de Gestión de Personas es una aplicación web desarrollada en PHP con el patrón de arquitectura MVC. Permite gestionar datos personales y realizar operaciones de Crear, Leer, Actualizar y Eliminar registros de manera eficiente.
